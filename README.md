# 🚀 Featured Project

## 🌳 Colcoor

**AI Conversation Orchestration Platform**

Colcoor is an event-driven platform for orchestrating AI-assisted research and software development. Instead of storing conversations as linear chats, it represents them as deterministic event tree, enabling structured exploration, reproducible reasoning, collaborative workflows, and efficient context management.

### Highlights

* Deterministic conversation tree
* AI workflow orchestration
* Collaborative AI research
* Branching conversations
* Context engineering
* Model Context Protocol (MCP)
* Production backend with FastAPI & PostgreSQL

**Repository**

https://github.com/Yosef-levy/colcoor
